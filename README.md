
## FOLHA DE PAGAMENTO

```markdown
# SISTEMA DE FOLHA DE PAGAMENTO

## Integrantes
- Integrante 1: GUILHERME
- Integrante 2: NICAEL
- Integrante 3: PEDRO

## Como executar

1. Compilar:
   ```
   javac src/*.java
   ```

2. Executar:
   ```
   java -cp src Main
   ```

## Funcionalidades
- Cadastrar colaborador (CLT, PJ, ESTAGIO)
- Listar todos os colaboradores
- Calcular folha de pagamento
- Sair do sistema

## Regras de cálculo
- CLT: R$ 2.000,00 + 10% = R$ 2.200,00
- PJ: R$ 2.000,00 - 11% = R$ 1.780,00
- ESTAGIO: R$ 2.000,00 - 20% = R$ 1.600,00
